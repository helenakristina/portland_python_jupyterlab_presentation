### Installation

- With <i><b>conda</b></i>

   ``` conda install -c conda-forge jupyterlab ```
   
- With <i><b>pip</b></i>

   ``` pip install jupyterlab ```
___   
### Starting Jupyter Lab

- In your terminal

   ``` jupyter lab ```
___
### Libraries you will need to reproduce the demo examples:
- pandas
- plotly
- geopandas
- descartes
- PySAL
